# Design Tokens (Figma → Tailwind)

## Colors

- Primary BG: Figma `Surface / Dark` → `bg-zinc-950`
- Card BG: Figma `Card` → `bg-zinc-950/60` + `border-zinc-800`
- Accent: Figma `Accent` → `text-sky-400` / `border-sky-400`
- Danger: Figma `Danger` → `text-red-400`

## Radii

- Cards & buttons: Figma `Radius 16–24` → `rounded-2xl`

## Typography

- H1: Figma `Display / Large` → `text-3xl md:text-4xl font-semibold`
- H2: Figma `Heading / Medium` → `text-lg md:text-xl font-semibold`
- Body: Figma `Body / Default` → `text-sm md:text-base text-zinc-300`
- Caption: Figma `Label / Small` → `text-xs text-zinc-500`
